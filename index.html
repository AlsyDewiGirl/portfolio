About Me

Hello, saya Alfi Syah Dewi Risgi

Saya Siswa Rekayasa Perangkat Lunak SMKN 2 Kota Bekasi dengan keinginan yang kuat terhadap teknologi dan pemecahan masalah. Saya memiliki minat yang besar dalam pengembangan web dan sistem TI, dan saya selalu berusaha untuk memperluas pengetahuan saya dengan menerapkannya pada proyek-proyek dunia nyata.

Selama saya PRAKERIN 2,5 bulan di PT. Elsewedy Electric Indonesia, Saya memperoleh pengalaman berharga di kedua bidang Warehouse dan Divisi IT. Kegiatan saya meliputi pengarsipan dokumen, mengganti nama dan memindai dokumen, serta berkontribusi pada proyek berbasis web seperti meningkatkan sistem peminjaman aset dan mengembangkan situs web manajemen pengunjung untuk tim keamanan. Pengalaman ini meningkatkan keahlian teknis saya sekaligus memperkuat kemampuan beradaptasi, kerja sama tim, dan ketelitian saya.

Saya berkomitmen pada pertumbuhan profesional yang berkelanjutan, dengan fokus membangun solusi yang efisien dan terukur sambil mengasah keterampilan saya dalam pengembangan perangkat lunak dan dukungan IT.

Services

Web Development

Projects

Project 1

Project ini membahas pengembangan sistem manajemen dan pengarsipan dokumen. Sistem ini dirancang untuk menyederhanakan proses pencatatan, penyimpanan, dan pengambilan dokumen sehingga dapat dikelola secara lebih efisien dan terorganisir.

Project 2

Project ini berfokus pada proses penggantian nama dokumen secara manual sesuai format yang telah ditentukan. Prosedur ini bertujuan untuk menjaga konsistensi dan standarisasi dalam penamaan berkas administratif, sehingga dokumen lebih mudah dikelola, dicari, dan dipindahkan ke penyimpanan jaringan warehouse.

Project 3

Project ini melibatkan pembangunan situs web berbasis PHP untuk pendaftaran tamu. Sistem ini mencatat data pengunjung, menyimpan riwayat kunjungan, mendukung integrasi webcam, dan menyediakan pencetakan slip tamu. Sistem ini dirancang dengan antarmuka pengguna modern dan basis data terintegrasi untuk manajemen tamu yang efisien.
